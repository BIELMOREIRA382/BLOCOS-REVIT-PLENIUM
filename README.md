## Adding Images
To add images to the Revit families catalog:
- Place the image files in the `images` directory of the repository.
- Reference the images in your Revit family documentation using the following markdown syntax:
  
  ![Alt text](images/your-image-file.jpg)

Ensure that your images are optimized for web use to improve loading times.

## Configuring GitHub Pages
To configure GitHub Pages for your catalog:
1. Go to the repository settings.
2. Scroll down to the "GitHub Pages" section.
3. Select the branch you want to use (typically `main`) and the root folder for the site.
4. Click "Save" and wait for GitHub to publish the site.

Your site will be available at `https://<username>.github.io/<repository-name>`.

## Information about the PORTA BASICA Families
The PORTA BASICA families are foundational elements in the Revit catalog. Here are some specifics:
- **Family Name:** PORTA BASICA
- **Description:** A basic door family used in various projects.
- **Parameters:** Width, Height, Material, etc.

Make sure to fill in the appropriate details for each family as you add them to the catalog.
